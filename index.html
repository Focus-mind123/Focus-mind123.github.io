Funciones principales

👨‍👩‍👦 Control parental

Las cuentas de menores pueden vincularse con la de un adulto para supervisar el estudio y el uso del dispositivo.

📅 Calendario de tareas

Los estudiantes pueden ver sus tareas organizadas por día para mantener un seguimiento claro de sus actividades.

⏱ Cronómetro inteligente

El cronómetro detecta cuando el teléfono está boca abajo para medir el tiempo real de estudio sin distracciones.

📊 Estadísticas de estudio

Gráficas semanales muestran cuánto tiempo se ha estudiado y cuánto se ha usado el celular.

📝 Notas y apuntes

Los estudiantes pueden escribir apuntes rápidos o tareas pendientes directamente en la app.

🚨 Botón de emergencia

Permite enviar la ubicación del menor al adulto vinculado si necesita ayuda.
